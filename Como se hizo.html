¿Cómo he realizado esta página web?

La página web la he realizado desde cero utilizando un editor de código de fuentes llamado. Visual Code (VS Code). Este software fue desarrollado por Microsoft.

Es un software libre y multiplataforma. Está disponible para Window, GNU/Linux y macOS.

La elección de este software lo basé porque cuenta con soporte para depuración de código, y dispone de múltiple número de extensiones, que básicamente te da la posibilidad de escribir y ejecutar código en cualquier lenguaje de programación.

Creación del código

Para la realización del código me he apoyado en un framework llamado Bootstrap. Este framework de desarrollo web es de código abierto y está diseñado para facilitar el proceso de desarrollo de los sitios web responsivos.

Bootstrap consta de scrips basados en HTML, CSS y JS y por tanto,podía demostrar mi dominio de habilidades en desarrollo web

Diseño de la web

Hice un estudio de experiencia de usuario, para que mi página web fuera ágil, ligera y legible.Buscaba que el usuario interactuara con diferentes enlaces, pero a la vez la información fuera clara y concisa

Otra de mis prioridades era un diseño actual y enfocado a mi profesión. Me apoyé en Canvas que es una herramienta de diseño gráfico, proporciona ideas y soportes y luego, en la personalización de sus productos,puedes demostrar tu valía y conocimientos en diseño web y diseño de interfaz

Para la personalización de estilos cree enlaces a las propiedades del framework. Para ello, usé el lenguaje CSS con un script donde desararrollé mis diseños.
