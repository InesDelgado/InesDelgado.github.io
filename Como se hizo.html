Bienvenido a mi página de explicación de código

Aquí explico cómo he realizado mi código y los conceptos detrás de él.

¿Cómo he realizado esta página web?

La página web se ha realizado utilizando un editor de código de fuentes llamado Visual Code (VS Code). Este software fue desarrollado por Microsoft.

Es un software libre y multiplataforma. Está disponible para Window, GNU/Linux y macOS.

Este software cuenta con soporte para depuración de código, y dispone de múltiple número de extensiones, que básicamente te da la posibilidad de escribir y ejecutar código en cualquier lenguaje de programación.

Creación del código

La realización del código de la página se asienta en un framework llamado Bootstrap. Este framework de desarrollo web es de código abierto y está diseñado para facilitar el proceso de desarrollo de sitios web responsivos.

Bootstrap consta de scrips basados en HTML, CSS y JS y por tanto, permite desarrollar una página web responsiva que se adapte a las necesidades del usuario o cliente final

Diseño de la web

La página se maquetiza tras un estudio de experiencia de usuario para que fuera atractiva y legible. El usuario interactúa con diferentes enlaces y opciones como desplegables o botones.

El diseño, colores y fondo son de elección personal. Canvas que es una herramienta de diseño gráfico, proporciona ideas y soportes para la personalización de sus productos

En personalización de estilos la página está hipervinculada por enlaces a las propiedades del framework. Los estilos de la página se apoyan en un scrip desarrollado en el lenguaje CSS
